# 贡献指南

## 文档状态

每篇文档建议使用以下状态：

| 状态 | 含义 |
| --- | --- |
| draft | 初稿，可继续补充 |
| reviewed | 已完成基础校对 |
| stable | 内容稳定，可作为学习主入口 |
| deprecated | 内容过时，仅保留迁移参考 |

## 写作模板

```md
# 标题

## 适合谁看

## 你会学到什么

## 前置知识

## 核心概念

## 基础示例

## 实战场景

## 常见问题

## 最佳实践

## 下一步学习
```

## 内容原则

- 优先解释业务含义和真实使用场景。
- 示例代码要能独立理解。
- 涉及版本差异时明确标注版本。
- 推荐资料优先引用官方文档。
- 影响项目结构、配置、路由或样式的变更，需要同步更新文档。
