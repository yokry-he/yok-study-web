# 速查手册总览

## 这个页面解决什么

速查手册不是完整教程，而是项目开发时的快速回查入口。它适合你已经学过某个技术，但写代码时需要快速确认 API、命令、写法、坑点和推荐选择。

如果你还没系统学过对应技术，先进入学习文档；如果只是忘了写法，再回到速查手册。

## 速查和学习文档的区别

| 类型 | 适合场景 | 阅读方式 |
| --- | --- | --- |
| 学习文档 | 第一次理解概念和项目场景 | 顺序阅读 |
| 速查手册 | 已经会用，但忘了写法或命令 | 按问题查 |
| 问题库 | 项目出错，需要定位根因 | 对照现象排查 |

不要把速查手册当作唯一学习材料。速查只告诉你“怎么写”，学习文档会解释“为什么这样写”和“项目里怎么取舍”。
