Network · Timing

Network · Cache source

判断顺序

1

2

3

最后结合文件名 hash 判断发布策略是否正确。

Playwright · Locator evidence

定位失败先比较测试意图与当前 DOM

错误日志告诉你“找不到什么”，DOM 快照告诉你“页面实际有什么”；两份证据必须对应同一时刻。

  • 需要判断文案变更、条件渲染还是加载未完成。
